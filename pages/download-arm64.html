Download Flow-Wing

Follow the steps below to install Flow-Wing on your system:

1. Install Homebrew if not already installed:

Visit https://brew.sh

2. If xcode CLI tools are not already installed, you can install the Xcode Command Line Tools using the command:

xcode-select --install

3. Tap the Flow-Wing repository:

brew tap kushagra1212/flow-wing

4. Install Flow-Wing:

brew install flowwing

5. Verify the installation:

flowwing --version
