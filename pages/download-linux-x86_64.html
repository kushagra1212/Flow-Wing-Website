Download Flow-Wing

Follow the steps below to install Flow-Wing on your system:

1. Download Flow-Wing Using wget:

wget https://github.com/kushagra1212/Flow-Wing/releases/download/v0.0.1-alpha.1/FlowWing-0.0.1-linux-x86_64.tar.gz

2. Extract the Archive:

tar -xzf FlowWing-0.0.1-linux-x86_64.tar.gz

3. Run the Installation Script:

cd FlowWing-0.0.1-linux-x86_64 && sudo bash install.sh

4. Verify the installation:

FlowWing --version
