Download Flow-Wing

Follow the steps below to install Flow-Wing on your system:

1. To download and run the install.sh or uninstall.sh script for Flow-Wing on a Linux system using a single wget command, use the following syntax:

wget -O - https://raw.githubusercontent.com/kushagra1212/Flow-Wing/main/scripts/prod/install-scripts/linux/install.sh | bash

For the uninstall.sh script:

wget -O - https://raw.githubusercontent.com/kushagra1212/Flow-Wing/main/scripts/prod/install-scripts/linux/uninstall.sh | bash
